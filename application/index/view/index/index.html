减少不必要的沟通，最大化开发效率：

工具亮点：
  1. 前后端开发人员能迅速找到需要的接口
  2. 移动端开发人员能请求任意后端开发人员的本地项目
  3. 移动端开发人员能对请求参数与响应数据一目了然。既兼具API文档查看功能
  4. 移动端开发人员能看到是谁最后修改的接口以请求对应开发人员的本地主机
  5. 可直接修改页面参数请求接口，不修改会以默认参数请求
  6. 返回参数每个都做备注未免浪费时间，因为很多参数一看就懂，可自定义备注响应数据，看自己心情
工具支持：
  1. HTTP请求，目前仅支持post和get
  2. 返回数据支持json，xml，html格式（目前仅对json格式做高亮处理）
